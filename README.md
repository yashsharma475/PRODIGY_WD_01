# PRODIGY_WD_01




## Interactive Navigation Menu

This project is submitted as **Task 01** for the **Prodigy InfoTech Web Development Internship**.

### 📌 Project Description
An interactive and responsive navigation menu built using HTML, CSS, and JavaScript.

The navigation bar:
- Remains fixed at the top of the page.
- Changes background color when the page is scrolled.
- Changes style when hovering over menu items.
- Includes smooth transitions for a better user experience.

### 🚀 Technologies Used
- HTML5
- CSS3
- JavaScript

### ✨ Features
- Fixed Navigation Bar
- Scroll Effect
- Hover Animation
- Responsive Layout
- Modern UI Design
